# Sobre o projeto

Este projeto é uma aplicação frontend desenvolvida com React utilizando o Vite como ferramenta de build.

O objetivo é servir como um template base para futuros projetos, consolidando conceitos fundamentais como:

- Componentização
- Gerenciamento de estado (State)
- Manipulação de eventos
- Roteamento com SPA
- Consumo de APIs
- Organização de código

----

# Tecnologias utilizadas
- React
- Vite
- Node.js (v20+)
- Axios
- React Router DOM
- Bootstrap

---

# Pré-requisitos

Antes de começar, você precisa ter instalado:

Node.js (versão 20 ou superior)
Gerenciador de pacotes (npm já vem com Node)

---
# Criando o projeto

```
npm create vite@latest nome-projeto
```

```
npm install
```
---
# Executando o projeto
```
npm run dev
```


---

# Instalando dependências

```
npm install axios react-router-dom bootstrap
```

---

# Limpando o projeto inicial

Apague o conteudo dos arquivos desnecessários gerados pelo template:

App.jsx
App.css
index.css

---

🎨 Configurando o Bootstrap

Importe o Bootstrap no arquivo main.jsx:
```
import 'bootstrap/dist/css/bootstrap.min.css';
```
